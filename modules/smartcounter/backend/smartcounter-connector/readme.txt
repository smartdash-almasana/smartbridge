== Description ==
Conector en tiempo real entre WooCommerce y SmartWork.

- Evita stock fantasma
- Detecta diferencias de conciliación
- Envía eventos automáticamente

== Installation ==
1. Descargar el plugin
2. Ir a WordPress > Plugins > Añadir nuevo
3. Subir archivo ZIP
4. Activar plugin

== Configuration ==
Editar el archivo PHP:

- SC_BACKEND_URL → tu backend
- SC_TENANT_ID → identificador del cliente
