# Product Scope

## Alcance inicial

El alcance inicial de SmartCounter es operacional y acotado:
- lectura de Excel contable caotico
- clasificacion de tipo de planilla
- mapping de columnas a campos canonicos
- normalizacion de datos con trazabilidad
- deteccion de ambiguedades para revision humana
- memoria operativa por cliente

## Usuarios objetivo

- estudio contable pequeno o mediano
- contador independiente

## Inputs esperados

- archivos Excel de clientes (uno o multiples)
- hojas con estructuras heterogeneas
- nomenclaturas no estandar y encabezados ambiguos

## Outputs esperados

- tipo de planilla detectado
- tabla util detectada por hoja
- campos canonicos mapeados
- dataset normalizado
- lista de ambiguedades/riesgos
- set de items para revision humana

## Limites del producto

- no reemplaza un sistema contable
- no ejecuta cierre contable integral
- no define reporteria financiera generalista
- no promete automatizacion total sin revision

## Prioridad comercial

Prioridad inicial: estudios contables y contadores independientes.
